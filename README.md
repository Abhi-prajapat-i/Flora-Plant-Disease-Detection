# 🌿 Flora

> **AI-Powered Plant Disease Detection & Treatment Advisor**

Flora is a deep learning based web application that detects plant diseases from leaf images and provides AI-generated treatment recommendations. The application uses dedicated CNN models for each crop to improve prediction accuracy and offers recommendations in multiple languages.

---

## 📌 Overview

Flora is designed to assist farmers, gardeners, and agriculture enthusiasts by providing quick and reliable plant disease diagnosis. Users simply select a crop, upload a leaf image, and receive disease predictions along with AI-powered treatment recommendations.

---

## ✨ Features

- 🌱 Multi-crop plant disease detection
- 🧠 Dedicated deep learning model for each crop
- 🤖 AI-powered treatment recommendations
- 🌍 Multi-language support

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/Abhi-prajapat-i/Flora-Plant-Disease-Detection.git
```

## Navigate to Project

```bash
cd flora-plant-disease-detection
```

## Create Virtual Environment

### Windows

```bash
python -m venv myenv
myenv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv myenv
source myenv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Application

```bash
streamlit run src/app.py
```

# 📖 How to Use

### Step 1

Select the crop.

### Step 2

Upload a clear leaf image.

### Step 3

Click **Predict Disease**.

### Step 4

The application displays:

- Plant Name
- Predicted Disease
- Confidence Score

### Step 5

Generate AI recommendations to receive:

- Disease description
- Symptoms
- Possible causes
- Chemical treatment
- Organic treatment
- Prevention tips
- Plant care recommendations
- Severity assessment
- Recovery guidance
